### Arch / Manjaro Linux

```
sudo pacman -S noto-fonts
```

###  Debian / Ubuntu / Linux Mint / Pop!_OS

```
sudo apt install fonts-noto
```

### Fedora

```
sudo dnf install google-noto-fonts
```

IFM - Lightweight Ncurses File Manager
======================================
Controls:
  [K/J]        Move up/down
  [Enter/L]    Open file/dir
  [Shift+J]    Move down a portion of files
  [Shift+K]    Move up a portion of files
  [H]          Go back
  [CTRL + H]   Toggle hidden files
  [Alt + H]    Go home
  [T]          Create file
  [M]          Create dir
  [Delete]     Delete file/dir
  [R]          Rename file/dir
  [C]          Mark/unmark file/dir
  [Shift+R]    Rename marked files/dirs
  [O]    Open with custom viewer
  [Q]          Exit
  [F1]         Help on command
  [I]          Open ABOUT, LICENSE, and COMMANDS help
  [g]          Go to ..
  [G]          Go to the last file




Launch commands:
  Usage:
     ifm [OPTION] 
     ifm [PATH]
  Examples:
     ifm                     Open current directory
     ifm Documents           Open 'Documents' directory
     ifm -h                  Show this help message
     ifm -?                  Open help menu

Mouse Controls:
  [Left Click]  Select/Open file/dir
  [Right Click] Go back
  [Scroll Up]   Move up
  [Scroll Down] Move down
